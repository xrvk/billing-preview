# Support

## How to file issues and get help

This project uses GitHub Issues to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new issue.

Do not attach real billing CSV files, screenshots with billing data, customer data, or any other sensitive information to public issues, pull requests, or discussions.

For help or questions about using this project, please file an issue.

**GitHub Copilot Billing Preview** is under active development and maintained by GitHub staff and the community. We will do our best to respond to support, feature requests, and community questions in a timely manner.

## GitHub Support Policy

Support for this project is limited to the resources listed above.
