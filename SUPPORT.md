# Support

## About this fork

xrvk's Billing Preview is an independent, unofficial fork of GitHub Copilot Billing Preview. It is not affiliated with or supported by GitHub, Inc. For support on the upstream project, see <https://github.com/github/copilot-billing-preview>.

## How to file issues and get help

This fork uses GitHub Issues on its own repository to track bugs and feature requests. Please search the existing issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new issue.

Do not attach real billing CSV files, screenshots with billing data, customer data, or any other sensitive information to public issues, pull requests, or discussions.

## Support policy

This project is maintained on a best-effort basis. There is no service-level commitment. Support is limited to the resources listed above.
